Core Capabilities

Engineered for Certainty.

🔍

Smart Snip Capture

Our custom snipping tool allows you to isolate specific posts, comments, or images without leaving your Facebook tab. Precision selection ensures the AI focuses exactly on the data you want to verify.

🛡️

Deepfake Analysis

Leveraging Pixtral's 400M parameter vision encoder, the extension detects lighting inconsistencies and unnatural textures common in AI-generated imagery (DALL-E, Midjourney, Flux).

⚖️

Contextual Fact-Checking

Goes beyond simple "True/False." The engine cross-references claims against known data to provide a nuanced verdict: True, Misleading, or Hoax—complete with a detailed explanation.

🌙

Adaptive UI

Built with modern CSS variables, the interface automatically syncs with your system or Facebook's Dark Mode for a seamless, eye-friendly experience during late-night browsing.

How to Install

1. Download the ZIP
2. Unzip the folder
3. Open chrome://extensions
4. Enable Developer Mode
5. Click Load Unpacked and select the folder.
6. When you open the Facebook page click the puzzle button at the top right and click the FactChecker extension.
